# 🎮 Purple Clicker Ultimate

Purple Clicker Ultimate, SGM (Sapanca Gelişim Merkezi) tarafından geliştirilen, yüksek performanslı ve hile korumalı (Anti-Cheat) modern bir web tabanlı tıklama oyunudur.

## 🚀 Özellikler
*   **Gelişmiş SGM Anti-Cheat:** Otomatik tıklama (auto-clicker) kullanan oyuncuları tespit eder ve 3 aşamalı uyarı sistemiyle cezalandırır.
*   **Bulut Senkronizasyonu:** Firebase entegrasyonu ile puanlarını ve yükseltmelerini güvenle sakla.
*   **20+ Yükseltme:** "Kırık Fare"den "SGM Tanrısı" seviyesine kadar uzanan geniş yükseltme ağacı.
*   **Retro-Modern Arayüz:** Mor (Purple) estetik ve neon efektli akıcı kullanıcı deneyimi.
*   **Global Sıralama:** Diğer geliştiriciler ve oyuncularla yarışabileceğin canlı liderlik tablosu.

## 🛠 Kullanılan Teknolojiler
*   **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+ Modules)
*   **Backend & DB:** Firebase (Auth & Firestore)
*   **Geliştirme Ortamı:** AndroidIDE & Termux

## 📥 Kurulum
Bu projeyi yerel ortamınızda çalıştırmak için:

1.  Depoyu klonlayın:
    ```bash
    git clone [https://github.com/efemert2181/purple-clicker.git](https://github.com/efemert2181/purple-clicker.git)
    ```
2.  Dosyaları bir web sunucusunda (veya Vercel/Netlify gibi platformlarda) çalıştırın.
3.  `firebaseConfig` kısmına kendi Firebase API anahtarlarınızı girdiğinizden emin olun.
